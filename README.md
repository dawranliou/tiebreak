# Tiebreak

A tennis game powered by [raylib][1] and its Common Lisp binding [cl-raylib][2]
through [cffi][3].

## Install

1. Follow instructions on [cl-raylib][2]
1. Clone this repo
1. Load `:tiebreak`
1. Run `tiebreak:main`

## Build

```
make build
```
